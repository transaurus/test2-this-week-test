## ⚛️ React {#react}

[![Next.js 15](/emails/issues/206/nextjs-15.png)](https://nextjs.org/blog/next-15)

[**Next.js 15**](https://nextjs.org/blog/next-15)

適逢 Next.js 大會，Next.js 15 正式發布。自 RC2 以來無重大變更，僅進行穩定性修復。本次 v15 重點：

- 快取行為反轉：多數功能（如 `fetch` 請求）預設不再快取
- App Router 採用 React 19 RC，Pages Router 維持 React 18
- 開發環境推薦使用 Turbopack（仍支援 Webpack）
- 含破壞性變更（如現有 API 改為非同步），可透過 `@next/codemod upgrade` 輔助升級

此外還包含大量開發體驗、安全性與效能改進。
