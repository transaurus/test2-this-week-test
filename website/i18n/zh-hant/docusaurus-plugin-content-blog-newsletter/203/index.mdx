## ⚛️ React {#react}

[![Server Actions 已更名為 Server Functions](/emails/issues/203/serverfunctions.jpg)](https://19.react.dev/reference/rsc/server-functions)

**[Server Actions 已更名為 Server Functions](https://19.react.dev/reference/rsc/server-functions)**

這不是程式碼變更，但卻是已正式發布在 React v19 文件網站的重要術語調整。Server Actions 現在被歸類為更廣泛的 Server Function 功能的子集。僅在以下情境中使用時才被視為 action：

- `<form action>`
- `useActionState()`
- `startTransition()`

此變更釐清了「use server」的核心是暴露 server 端點，不限於變更操作。使用 Server Functions 獲取資料和 JSX 可能不再是反模式。

**關於 Next.js Server Actions**：它們使用 `POST` 請求並順序執行，但這是 Next.js 的實作細節。React 並不禁止並行執行 actions。Next.js 團隊已確認將引入新 API，支援以 `GET` 方式呼叫 Server Functions。

**[React DevTools 6.0](https://github.com/facebook/react/blob/main/packages/react-devtools/CHANGELOG.md#600)**
