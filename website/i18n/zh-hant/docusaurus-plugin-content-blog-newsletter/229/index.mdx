## 📱 React-Native {#react-native}

[![React Native 0.79](/emails/issues/229/react-native-0.79.png)](https://reactnative.dev/blog/2025/04/08/react-native-0.79)

**[React Native 0.79 – 更快的工具及更多功能](https://reactnative.dev/blog/2025/04/08/react-native-0.79)**

近期 React Native 核心團隊轉向更頻繁的發布週期，儘管這是較小的版本，但在效能改進方面仍令人振奮。重點如下：

- Metro：3 倍速冷啟動及套件匯出支援
- JavaScriptCore (JSC) 移至社群套件
- iOS：Swift 相容的原生模組註冊
- Android：藉由跳過 JS 套件壓縮，實現更快的應用啟動
- 移除遠端 JS 除錯功能
