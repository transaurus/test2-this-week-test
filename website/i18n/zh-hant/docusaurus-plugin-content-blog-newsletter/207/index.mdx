## 📱 React-Native {#react-native}

[![預設啟用新架構、開發者工具與更多功能](/emails/issues/207/newarch.jpg)](https://reactnative.dev/blog/2024/10/23/release-0.76-new-architecture)

[**React Native 0.76 - 預設啟用新架構、開發者工具與更多功能**](https://reactnative.dev/blog/2024/10/23/release-0.76-new-architecture)

React Native 0.76 發布重大更新！新架構現已預設啟用，新增 React Native DevTools 強化除錯能力，並支援 CSS 的 `boxShadow` 和 `filter` 屬性以提升 UI 效果（僅新架構可用）。請注意破壞性變更：最低 SDK 需求已提高至 iOS 15.1 和 Android SDK 24（Android 7）。立即升級體驗！

v0.76 雖酷，但更棒的是[新架構已正式宣告](https://reactnative.dev/blog/2024/10/23/the-new-architecture-is-here)可投入生產環境！這是歷經多年的重大里程碑！您可能已聽聞 JSI、Fabric、Turbomodules、同步低開銷無橋接通訊、互操作模式等技術。這不僅關乎效能，更解決了諸如同步測量視圖定位工具提示、讓 `useLayoutEffect` 如網頁般運作等 UX/DX 痛點。最令人振奮卻被低估的是：React Native 終於能完整運用 React 18+ 的並行功能，特別是過渡效果、Suspense 及未來新特性。
