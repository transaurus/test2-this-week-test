## 📱 React-Native {#react-native}

本週 React Native 新聞由 Benedikt 編譯。如有回饋、建議或新聞分享？請透過 [Twitter](https://twitter.com/bndkt) 聯繫我。

[![Expo SDK 50](/emails/issues/171/expo50.jpg)](https://expo.dev/changelog/2024/01-18-sdk-50k)

**[Expo SDK 50](https://expo.dev/changelog/2024/01-18-sdk-50)**

Expo SDK 50 引入了 Expo 開發工具插件，新增 SQLite 和 Camera API，並加入 `@expo/fingerprint`——這是一種判斷更新後的 JS 套件是否與特定原生版本相容的新方法。此版本也標誌著 Expo 中 [Webpack 支援的終結](https://blog.expo.dev/webpack-support-in-expo-cli-is-now-deprecated-e9831d7eb631)，現已棄用並轉向 Metro。了解 Expo 50 最快速便捷的方式，可能是觀看 Simon 在 [YouTube](https://www.youtube.com/watch?v=shaGaiwu4Fg) 上約 15 分鐘的快速解說。

[![Expo Router v3](/emails/issues/171/router3.jpg)](https://expo.dev/changelog/2024/01-23-router-3)

**[Expo Router v3](https://expo.dev/changelog/2024/01-23-router-3)**

隨 Expo SDK 50 同步發布的 Expo Router v3 帶來大量效能與穩定性改進，並提供更完善的文件。最值得注意的是，現已實驗性支援透過 API Routes 建立通用伺服器端點。Expo Router 現已成為首個真正的通用全端 React 框架！
