## ⚛️ React {#react}

[![React Labs - 我們二月份的工作進展](/emails/issues/174/react-labs.jpg)](https://react.dev/blog/2024/02/15/react-labs-what-we-have-been-working-on-february-2024)

**[React Labs - 我們二月份的工作進展](https://react.dev/blog/2024/02/15/react-labs-what-we-have-been-working-on-february-2024)**

本週我們迎來了 React Labs 自幾乎整整一年以來的首次官方更新，他們發布了一些令人興奮的公告。我們在這裡已經涵蓋了其中許多內容，但有一份官方總結總是好的！

React Forget 🐦 [現在只叫做 React Compiler](https://twitter.com/en_JS/status/1758224340880158955)，而且似乎在生產環境 🐦 [Instagram 的部署](https://twitter.com/sebastienlorber/status/1734607171030303186)已經成功。

下一個 React 版本將會是主要版本。React 19 即將來臨，而且有個特定的[會議](https://conf.react.dev/)非常適合用來宣布這樣的訊息 😏。此版本還將內建支援管理頁面元數據（內建了 React-Helmet 為我們提供的功能，並具有適當的串流支援）、資源載入（包括 Suspense 整合）以及用於從客戶端發送資料到伺服器的 Actions。像 Next.js 和 Remix 這類元框架需要以專有實作涵蓋的範圍正在縮小。

其中一個最被低估的即將推出的 React 功能，Offscreen API（命名為 Activity），為了專注於 React 19 而被暫時降低優先級。我們必須再等待一段時間，才能充分利用 React 18 引入的並行模型。

總體而言，現在是成為 React 開發者的激動時刻，我們期待下一個主要版本！

[![React Strict DOM](/emails/issues/174/strict-dom.jpg)](https://github.com/facebook/react-strict-dom)

**[React Strict DOM](https://github.com/facebook/react-strict-dom)**

本週 Meta 不僅發布了 React 的重大消息。今日 GitHub 上出現了一個全新儲存庫：[react-strict-dom](https://github.com/facebook/react-strict-dom)。README 將其描述為「React DOM 與 StyleX 的實驗性整合，旨在改進並標準化網頁與原生平台的樣式化 React 元件開發」。其基於 [「React DOM for Native」RFC](https://github.com/react-native-community/discussions-and-proposals/pull/496) 的概念，我認為這將對未來通用應用開發的思維方式產生重大影響。目前 React Native 實現通用應用的主流方案是 React Native Web（RNW），它將 RN 元件與 API 墊片移植到網頁平台。儘管 RNW 非常優秀，但常讓人感覺像是將某個平台強行套入另一個平台。react-strict-dom 的核心前提是建立一個統一支援網頁、iOS 和 Android（未來可能擴展更多平台）的單一 DOM 函式庫。其元素 API 基於網頁標準，因此您會在上方截圖中看到類似 HTML 的語法。

一如往常，Dan Abramov 最擅長用簡潔的 🐦 [推文](https://twitter.com/dan_abramov2/status/1760245165703184516) 表達概念：

> 「這看起來像是 RNW 的精神繼承者，但採用了相反方向（網頁 API 已成主流，因此 RN 正建立對嚴格子集的支持）。本質上，您將能在 RN 中渲染 `<div>`。」

檢視目前的[相容性狀態](https://github.com/facebook/react-strict-dom/blob/main/packages/react-strict-dom/COMPATIBILITY.md)可知仍有長路要走，但這無疑是邁向更通用 React 的重要里程碑！Meta 已在內部使用此方案！另參見 Lorenzo 的 🐦 [討論串及微軟的參與](https://twitter.com/Kelset/status/1760367655423549480)。
