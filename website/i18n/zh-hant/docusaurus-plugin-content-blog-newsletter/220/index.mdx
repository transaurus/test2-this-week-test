## ⚛️ React {#react}

[![Next.js PR - 支援 Node.js 中介軟體](/emails/issues/220/nextjs-middleware.jpg)](https://github.com/vercel/next.js/pull/75624)

**[Next.js PR — 支援 Node.js 中介軟體](https://github.com/vercel/next.js/pull/75624)**

A much-awaited feature for the Next.js community. The latest v15.2 canary has a new `experimental.nodeMiddleware` flag letting you opt-in for `runtime: ‘nodejs’`, enabling you to use Node.js APIs in middleware, and remove the constraints of being limited to APIs compatible with the edge runtime.

Vercel 中介軟體也已[徹底重建](https://x.com/galstar/status/1886832573482934706)，支援多種運行時並採用[流體計算(Fluid compute)](https://vercel.com/blog/introducing-fluid-compute)，這是 Vercel 運行無伺服器函數的新成本效益方案。
