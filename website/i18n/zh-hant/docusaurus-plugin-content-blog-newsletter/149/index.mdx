## ⚛️ React {#react}

[![Panda CSS](/emails/issues/149/panda.jpg)](https://panda-css.com/)

**[Panda CSS](https://panda-css.com/)**

Segun Adebayo（Chakra UI 創建者）推出期待已久的 CSS-in-JS 函式庫（[預告](https://twitter.com/thesegunadebayo/status/1669732623994834946)）。具備卓越開發體驗，基於現代 CSS 特性（層疊圖層），高效能（建置時靜態分析、無運行時、原子化 CSS）... 簡言之，滿足多項關鍵需求並引發高度關注。特別與 React 整合良好（JSX 樣式屬性、相容 React Server Components 等），亦支援其他框架。

這是值得密切關注的函式庫，有望與 Tailwind 競爭：能更佳整合目標框架，且無需強制使用冗長 `className`。
