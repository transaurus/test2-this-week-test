## ⚛️ React {#react}

[![](/emails/issues/150/nextjs-update.jpg)](https://nextjs.org/blog/june-2023-update)

**[Next.js App Router 更新 - 優先事項](https://nextjs.org/blog/june-2023-update)**

Next.js 團隊說明新版 App Router 的開發背景。他們近期收到大量批評，並清楚知曉當前開發體驗存在的錯誤與問題。重點工作方向如下：

- 效能：本地環境、建置與伺服器運行階段
- 錯誤修復與穩定性提升
- 教學資源與文件優化

[![](/emails/issues/150/skew-protection.jpg)](https://vercel.com/blog/version-skew-protection)

**[Vercel - 版本偏移防護機制](https://vercel.com/blog/version-skew-protection)**

Vercel 新功能，解決新部署時可能發生的前後端不同步問題：狀態化客戶端請求可被導向對應的靜態伺服器部署版本。特別適用於持續部署場景。Next.js 13.4.7 已實驗性支援，將自動附加正確的 HTTP 標頭。

這套 Tailwind + Radix 元件組合正逐漸受到關注。作者最近發佈了一個重大更新，其中包含一個已經提供出色開發者體驗（DX）的 CLI 工具，並預計在不久的將來加入一些[令人印象深刻的 AI 功能](https://twitter.com/shadcn/status/1673055193968287744)。Kent C. Dodds [解釋](https://twitter.com/kentcdodds/status/1673806183935909888)了為何在其 Remix 技術堆疊中採用此方案。
