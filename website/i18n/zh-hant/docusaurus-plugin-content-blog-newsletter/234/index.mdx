## ⚛️ React {#react}

[![TanStackDB](/emails/issues/234/tanstackdb.jpg)](https://github.com/TanStack/db)

**[TanStack DB](https://github.com/TanStack/db)**

TanStack 團隊與 ElectricSQL 合作推出了一個全新的函式庫，帶來許多令人興奮的功能。它提供了一個極快的查詢引擎、細粒度的響應性以實現最少的重新渲染、穩健的交易原語以及預設的標準化數據！

它建立在 TanStack Store 之上，並與 TanStack Query 協同工作，透過集合、即時查詢和樂觀更新擴展了其功能。

## 📱 React-Native {#react-native}

[![MS Office uses React Native](/emails/issues/234/microsoft-uses-react-native.png)](https://devblogs.microsoft.com/react-native/2025-05-09-office-modernize/)

**[Office 如何利用 Windows App SDK 和 React Native 現代化其應用套件的 UI](https://devblogs.microsoft.com/react-native/2025-05-09-office-modernize/)**

微軟的 Chiara Mooney 描述了他們如何使用 React Native 在旗艦級 MS Office 套件中實現最新的 Copilot 功能，該套件被 **6 億人**使用。

他們將其與島嶼架構（island architecture）結合使用，這是前端生態系統中由 Astro 等框架採用的模式。

有趣的是看到 RN 如何在不僅限於 Android 或 iOS 的平台上，於最高企業級別中被應用。
