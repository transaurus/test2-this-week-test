## ⚛️ React {#react}

**[React 我愛你，但你讓我失望了](https://marmelab.com/blog/2022/09/20/react-i-love-you.html)**

React-Admin 作者針對 React 多個面向（表單、useContextSelector、refs、依賴陣列...）提出中肯批評。別錯過 [Dan Abramov 的回應](https://twitter.com/dan_abramov/status/1572590212647391235)。

**[5 分鐘了解 Next.js 佈局 RFC](https://vercel.com/blog/next-js-layouts-rfc-in-5-minutes)**

Next.js 基於檔案系統的路由器將在下個主要版本大幅升級。新增支援嵌套佈局（類似 Remix），可避免瀏覽時不必要的佈局卸載/重載，同時管理資料載入與錯誤處理。本文提供快速概覽，另見 [Delba 的推文](https://twitter.com/delba_oliveira/status/1568291558776274945)。

**[React Router 6.4](https://remix.run/blog/react-router-v6.4)**

重大更新：如預期，許多 Remix API（資料載入、變更...）最終遷移至 React-Router。觀看 [2 分鐘影片](https://www.youtube.com/watch?v=aX74DbFT5nI)

**[Gatsby 2022 有哪些新功能？](https://www.smashingmagazine.com/2022/09/new-gatsby-2022/)**

Paul 將其部落格升級至最新 Gatsby v4 版本，並分享使用最新功能的經驗。延遲靜態生成能顯著減少建置時間。另見 [Gatsby 5 總體討論](https://github.com/gatsbyjs/gatsby/discussions/36609) 與影片 [Gatsby v5 Alpha 展示](https://www.youtube.com/watch?v=C-WrnfUm33k)：局部 hydration 即將來臨！
