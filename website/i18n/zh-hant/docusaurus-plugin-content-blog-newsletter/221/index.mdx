## ⚛️ React {#react}

[![State of React 2024 - 調查結果](/emails/issues/221/state-of-react.png)](https://2024.stateofreact.com)

**[State of React 2024 - 調查結果](https://2024.stateofreact.com)**

State of React 2024 的結果已經出爐。這是一個資訊金礦，我們無法在此談論所有內容，但以下是一些重點：

- 在「新」的 React 功能中，伺服器元件最受歡迎，約 7000 名受訪者中有 40% 表示「使用過」，這可能要歸功於 Next.js……
- `forwardRef` 是最不受歡迎的 React API，而且即將消失！
- 你聽說過 React 的 Taint API 嗎？看起來有 80% 的 React 開發者沒聽過，這可以理解，因為它還是實驗性的。
- 排名第一的位置變化不大：Next.js、TanStack Query、Axios、Jest 和 Zod 仍然是最常用的工具。
- 一些新星：Zustand 和 Vitest 的使用者增加了，TanStack Start 在興趣度上排名第一，儘管幾乎還沒有人使用過它。

Seb 撰寫了[調查結論](https://2024.stateofreact.com/en-US/conclusion/) — 記得去看看！🙂
