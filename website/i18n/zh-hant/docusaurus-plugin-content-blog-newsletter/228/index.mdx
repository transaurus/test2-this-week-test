## ⚛️ React {#react}

[![React 19.1](/emails/issues/228/react_19_1.jpg)](https://github.com/facebook/react/releases/tag/v19.1.0)

**[React 19.1](https://github.com/facebook/react/releases/tag/v19.1.0)**

本週發布的 React 19.1 最值得關注的變更包括：

- 新增 [`captureOwnerStack()`](https://react.dev/reference/react/captureOwnerStack) API，供框架開發者用於在開發環境提供更完善的錯誤資訊。其處理 `children` 和 DOM 節點的方式有別於現有的元件堆疊追蹤機制
- Parcel 的伺服器元件支援已整合至 React 端，需搭配 Parcel 2.14 的新功能使用
- 實驗性功能 `unstable_prerender()`，用於在伺服器端預渲染 React 伺服器元件

此版本還包含大量次要變更與修復，多數涉及伺服器元件或 `Suspense` 邊界處理
