## ⚛️ React {#react}

**[Vercel OG 圖片生成：在 Edge 上快速生成動態社群卡片圖片](https://vercel.com/blog/introducing-vercel-og-image-generation-fast-dynamic-social-card-images)**

Vercel 提出了一種在 Vercel Edge Functions 中生成動態社群卡片的非常高效的解決方案。實用性：可以在無需額外配置的情況下使用 JSX、React 元件（無狀態）和 Tailwind。我推薦 [這篇推文串](https://twitter.com/shuding_/status/1579607964549513217)，解釋了整個運作方式，非常有趣！他們甚至使用了 React-Native 的佈局引擎 Yoga 🤭

**[React 中渲染的未來](https://prateeksurana.me/blog/future-of-rendering-in-react/)**

這篇圖文並茂的文章介紹了 React 18 及更高版本的渲染模式。您將找到關於 SSR 串流如何運作、Suspense、React lazy、以及使用 Server Components 實現功能標誌的資訊...

**[克服 React 專案中的常見問題](https://dinosaurs-with-jetpacks.com/posts/react-problems)**

鼓勵將我們的 JavaScript 應用程式與框架（React 或其他）解耦。提倡回歸更簡單的方式。一些可能會引起爭議的觀點。

**[Storybook 中的 Material UI](https://storybook.js.org/blog/material-ui-in-storybook/)**

本教程有一個特別有趣的章節，解釋了如何透過重複使用底層 UI 函式庫的 TypeScript 類型來自動建立 Storybook 控制項。在轉發 props 時非常有用。甚至支援像 `Pick` 和 `Omit` 這樣的輔助類型。
