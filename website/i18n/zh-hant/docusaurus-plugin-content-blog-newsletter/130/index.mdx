## ⚛️ React {#react}

[![Next.js 13.1](/emails/issues/130/nextjs-13.1.jpg)](https://nextjs.org/blog/next-13-1)

**[Next.js 13.1](https://nextjs.org/blog/next-13-1)**

此次小版本更新包含多項改進：

- 應用程式目錄(app directory)改進與錯誤修復
- TypeScript 插件，提升開發體驗
- 模組轉譯功能
- Turbopack 與效能基準測試改進
- 用於轉換導入的 SWC 插件
- API 路由的 Edge 運行時趨於穩定，並支援進階中介軟體

- 📖 Beta docs [useEffectEvent](https://beta.reactjs.org/learn/separating-events-from-effects#declaring-an-effect-event): some updated doc for the newly renamed React hook. The [new site is coming in January](https://twitter.com/dan_abramov/status/1606646280649904131)!

## 📱 React-Native {#react-native}

[![Tamagui 1.0](/emails/issues/130/tamagui.jpg)](https://tamagui.dev/blog/version-one)

**[Tamagui 1.0](https://tamagui.dev/blog/version-one)**

Tamagui 已正式結束測試階段，讓您能輕鬆基於 React 創建跨平台專案（網頁與行動裝置），使用相同程式碼庫的同時，不犧牲開發體驗或效能。專案分為三大模組：

- Tamagui Core：基於 React-Native 和 [React-Native-Web-Lite](https://www.npmjs.com/package/react-native-web-lite) 的 CSS-in-JS 運行環境
- Tamagui Static：支援視圖扁平化與原子化 CSS 樣式提取的 Babel 編譯器及 Node 預評估工具
- Tamagui UI：提供易用 React 元件與鉤子的函式庫

Tamagui 是創新專案，獲得樣式與跨平台領域思想領袖的正面評價。即使您僅針對單一平台開發也值得嘗試。其已支援 React 18 功能如伺服器元件、並行模式。強烈建議體驗！

[![React Native Skia: 年度回顧與未來展望](/emails/issues/130/skia.jpg)](https://shopify.engineering/react-native-skia-2022)

**[React Native Skia: 年度回顧與未來展望](https://shopify.engineering/react-native-skia-2022)**

William Candillon 對 React-Native-Skia 專案進行了精彩回顧。[Skia](https://skia.org/) 是 Google 的 2D 繪圖引擎，被 Chrome、Android、Flutter 等採用。團隊透過 UI Thread Rendering 技術與全新聲明式 Skia DOM API 顯著提升效能。該函式庫具擴展性（如 Skottie、VisionCamera 影格處理）。2023 年預計將推出基於 worklet 的動畫、與 Reanimated 的深度整合，以及文字佈局強化功能。
