## ⚛️ React {#react}

[![React 伺服器元件開發者工具](/emails/issues/155/rscdevtools.jpg)](https://www.alvar.dev/blog/creating-devtools-for-react-server-components)

[**React 伺服器元件開發者工具**](https://www.alvar.dev/blog/creating-devtools-for-react-server-components)

隨著 React 伺服器元件(React Server Components)的發布，Alvar 注意到客戶端與伺服器間的網路通訊使用了一種難以解析的格式。因此他開發了網站工具 [RSC 解析器](https://rsc-parser.vercel.app/)，能更便捷地檢視數據負載。雖然有效，但用戶體驗並不理想。此後他轉向開發瀏覽器擴充套件，功能類似 Chrome 的網路開發者工具，具備時間軸顯示即時數據流，甚至支援 Suspense 串流處理。該工具已協助他偵測到部落格中過大的數據負載並進行優化。

這是個頗具潛力的開發中工具，值得持續關注。擴充套件尚未上架瀏覽器商店，但即將推出。可透過此 [Twitter 討論串](https://twitter.com/sebastienlorber/status/1689203357368365056)觀看操作示範影片。
