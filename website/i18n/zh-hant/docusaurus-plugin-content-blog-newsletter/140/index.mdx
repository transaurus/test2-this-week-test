## ⚛️ React {#react}

[![React 實驗室](/emails/issues/140/reactlabs.jpg)](https://react.dev/blog/2023/03/22/react-labs-what-we-have-been-working-on-march-2023)

**[React 實驗室：2023 年 3 月工作進展](https://react.dev/blog/2023/03/22/react-labs-what-we-have-been-working-on-march-2023)**

來自 React 研發團隊的官方更新，涵蓋多項主題：

- React Server Components：一個優秀的實作已可透過 Next.js 的 App Router 使用。具備 async/await 的 RSC 也應能在客戶端應用中運行。我感覺他們希望統一客戶端/伺服器端模型，我認為 [AsyncContext](https://twitter.com/sebastienlorber/status/1639183216799100928) 對此會非常理想！
- 資源載入：與 Suspense 整合以載入 CSS、字型和圖片（[更多細節](https://twitter.com/acdlite/status/1637102880850059264)）。
- 文件元資料：為 `<head>` 元資料提供具串流支援的一級 API。
- React 優化編譯器：不再定位為自動記憶化編譯器，而是反應式編譯器。[React Forget 已達成里程碑](https://twitter.com/en_JS/status/1641494974893281282)，團隊正專注整合工作。
- 離屏渲染：這項創新令我非常期待，它使 React 有別於其他框架。Meta 正在 React-Native 應用上測試預渲染，已獲積極成果。
- 過渡追蹤：可偵測緩慢的過渡狀態。
