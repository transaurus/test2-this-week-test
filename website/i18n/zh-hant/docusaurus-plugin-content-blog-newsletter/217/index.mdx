## ⚛️ React {#react}

[![揭秘：React 實驗性動畫 API](/emails/issues/217/viewtransition.jpg)](https://motion.dev/blog/reacts-experimental-view-transition-api)

**[揭秘：React 實驗性動畫 API](https://motion.dev/blog/reacts-experimental-view-transition-api)**

Motion 創建者解析使用視窗轉場(View Transitions)的動畫取捨，以及為何 React 需通過新的 `<ViewTransition>` 元件將其直接整合至協調器。這篇互動式文章展示了即將推出的 React 功能早期樣貌，包含當前實作細節，並解釋其與 Motion 佈局動畫的部分重疊性。

請注意這項令人振奮的新 API 仍在積極開發中，可能變動，且持續有新 PR 合併，例如：

- 👀 [React Core PR - 僅支援 View Transitions v2](https://github.com/facebook/react/pull/31996)：新元件將利用 v2 規範 API（如 `view-transition-class`），需 Safari 18.2+，Firefox 尚不支援。
- 👀 [React Core PR - 基於事件類型的視窗轉場類別名稱](https://github.com/facebook/react/pull/32050)
- 👀 [Next.js PR - `experimental.viewTransition` 標記](https://github.com/vercel/next.js/pull/74659)：可在 Next.js 15.2 canary 4 試用此功能。
