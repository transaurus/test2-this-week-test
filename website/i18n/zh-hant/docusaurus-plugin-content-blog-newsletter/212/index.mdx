## ⚛️ React {#react}

[![Astro 5.0](/emails/issues/212/astro.jpg)](https://astro.build/blog/astro-5/)

**[Astro 5.0](https://astro.build/blog/astro-5/)**

Astro 剛剛發布了重大更新。雖然技術上不是純 React 框架，但仍在 React 社群廣泛使用，可輕鬆整合互動式 React 元件，並依賴 MDX。重點包括：

- Content Layer：將所有內容整合到友善、型別安全的資料儲存庫，包含來自 CMS、API 和其他來源的內容
- Server Islands：盡快提供初始頁面回應，同時保持將動態內容嵌入靜態布局的能力。這是 React Server Components 和 Next.js 部分預渲染的有趣簡化替代方案
- 簡化預渲染，移除了「混合」選項
- 型別安全的環境變數
- Vite 6 搭配全新 Environment API
- 實驗功能：圖片裁剪、響應式圖片布局、SVG 元件
