## ⚛️ React {#react}

[![深入解析 Framer 的 Magic Motion](/emails/issues/124/framer-motion.png)](https://www.nan.fyi/magic-motion)

**[深入解析 Framer 的 Magic Motion](https://www.nan.fyi/magic-motion)**

一篇出色的互動文章，解釋了佈局動畫的運作原理。詳細解構了 FLIP 技術，該技術透過 CSS 變換（位置、縮放）使這類轉場效果更高效。我們逐步重現了 Framer Motion 的佈局動畫系統。

關於相同主題，也推薦閱讀 [關於 Framer Motion 佈局動畫的一切](https://blog.maximeheckel.com/posts/framer-motion-layout-animations/)。

[![Next.js 13](/emails/issues/124/nextjs13ts.jpg)](https://twitter.com/delba_oliveira/status/1592209522281439232)

**[Next.js 13 TypeScript 插件](https://twitter.com/delba_oliveira/status/1592209522281439232)**

舒丁（Shu Ding）正在開發一款 TypeScript 語言服務插件（[原始碼](https://github.com/vercel/next.js/blob/canary/packages/next/server/next-typescript.ts)），透過提供文件說明、新錯誤提示及針對 Next.js 與 Server Components 慣例的自動補全功能，大幅提升 Next.js 開發體驗。這項工具巧妙地彌補了 TypeScript 中[無法為 ES 模組匯出內容添加類型](https://github.com/microsoft/TypeScript/issues/38511)的限制——此功能廣受框架開發者期待。

[![Writing Performant CSS with vanilla-extract](/emails/issues/124/vanilla-extract.png)](https://www.lekoarts.de/javascript/writing-performant-css-with-vanilla-extract)

**[使用 vanilla-extract 編寫高效能 CSS](https://www.lekoarts.de/javascript/writing-performant-css-with-vanilla-extract)**

一篇關於 [vanilla-extract](https://vanilla-extract.style/) 的詳盡文章，這是由 Mark Dalgleish（CSS Modules 創建者）開發的無運行時 CSS-in-JS 函式庫。同時介紹其原子化 CSS 層 [Sprinkles](https://vanilla-extract.style/documentation/packages/sprinkles/)。範例基於 React：重現 Tailwind UI 元件。大量比較 Tailwind 與其他 CSS-in-JS 方案。誠然，它功能強大但並不簡單😅 我們需要更深入研究。
