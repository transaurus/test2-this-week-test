## ⚛️ React {#react}

[![React 19](/emails/issues/213/react.jpg)](https://react.dev/blog/2024/12/05/react-19)

**[React 19](https://react.dev/blog/2024/12/05/react-19)**

過去一年 React 19 RC 版始終是本通訊焦點，如今正式穩定版除新增兩項功能（延遲發布的預熱機制和 React-DOM 靜態 API）外，其餘已涵蓋功能包括：

- Actions / Server Actions / `useActionState()`
- `<form>` + 漸進增強相容的鉤子
- 使用 `<Context>` 取代 `<Context.Provider>`
- `use()` 鉤子
- 自訂元素
- Ref 作為標準屬性及 ref 回調清理函式
- 元標籤處理
- 資源預載
- 更完善的錯誤與水合差異報告
