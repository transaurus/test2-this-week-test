## 📱 React Native {#react-native}

[![2024 年 React Native 狀態調查結果](/emails/issues/223/state-of-rn.jpg)](https://results.stateofreactnative.com/)

**[2024 年 React Native 狀態調查結果](https://results.stateofreactnative.com/)**

調查結果已經出爐，我感到非常自豪的是，在所有的 React Native 資源中，這份通訊獲得了最高的正面評價分數。謝謝大家！❤️

我無法總結整個調查，但以下是我發現的一些重點：

- 61% 的 RN 開發者來自 React（網頁）
- iOS 和 Android 仍然是最常見的 RN 目標平台，但網頁也佔有 22% 的市場份額
- 獲得最多正面評價的函式庫有：Reanimated、TanStack Query、Zustand、React Navigation、Expo Router、Skia。
- 在樣式方面沒有明顯的贏家。開發者喜歡內建的 API，例如 StyleSheet 和內聯樣式。
