大家好！

本週我們有一些非常棒的 React 文章，同時也有來自 Waku 和 Remix 的有趣版本發布。社群針對 React Server Components 的複雜性和所有權問題，分享了經過緩和但有趣的回饋。

在 React Native 方面則較為平靜。我們正熱切等待 Expo SDK 50 的發布，這次的正式版比測試版晚了很多。我們也在等待 VisionOS 的正式推出，這是一個 React Native 將支援的新平台。

我剛剛[慶祝了 10 年的 React 經驗](https://twitter.com/sebastienlorber/status/1747307299356447148) 🙉 很難相信我的職業生涯因為 React 而改變，尤其考慮到我以前真的很不喜歡 JavaScript。

## ⚛️ React {#react}

[![Waku 0.19](/emails/issues/170/waku.jpg)](https://waku.gg/blog/introducing-create-pages)

**[Waku 0.19](https://waku.gg/blog/introducing-create-pages)**

此版本引入了新的 `createPage` API，用於以程式設計方式建立佈局和頁面。這標誌著 Waku 從一個 React Server Components 的學習體驗轉變為真正的生產就緒 React 框架。預計在 v0.20 中會有一個輕量級的基於檔案的路由。
