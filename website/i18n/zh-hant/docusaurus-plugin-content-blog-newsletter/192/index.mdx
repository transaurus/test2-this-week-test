## ⚛️ React {#react}

[![Remix - 戰爭迷霧](/emails/issues/192/rr.jpg)](https://remix.run/blog/fog-of-war)

**[Remix - 戰爭迷霧](https://remix.run/blog/fog-of-war)**

我們上期已介紹過此功能，現在有篇專文詳細解釋其原理。這項 React Router 新功能可將路由清單分割成區塊。本文說明它如何整合到 Remix 2.10 及相關取捨。事實證明路由清單可能變得相當龐大，若不分割，可能因需預先下載而延遲應用程式的 React hydration。若您擁有中大型 Remix 或 React Router 網站，建議立即檢查路由清單大小，並規劃邏輯分割方式，因為此功能帶來的體積縮減能顯著提升效能。Shopify 的初始清單從 10MB 降至 1.9KB！Docusaurus 網站清單也有 200KB，我絕對計劃實作此功能。

[![react-native-filament](/emails/issues/192/filament.jpg)](https://github.com/margelo/react-native-filament)

**[react-native-filament](https://github.com/margelo/react-native-filament)**

Margelo 團隊發布了基於 Google Filament 的全新實時物理 3D 渲染引擎，適用於 React Native。它通過 Metal/Vulkan 進行 GPU 加速，在獨立線程渲染，兼容 Reanimated，並已支持新架構和 Expo。README 詳細解釋了它與基於 expo-gl 的現有 3D 解決方案（如 expo-three 和 react-three-fiber RN）的差異。隨著 Skia 和 Filament 的發展，React Native 的 2D/3D 能力前景光明。
