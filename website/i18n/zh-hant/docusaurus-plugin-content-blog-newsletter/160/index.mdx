## 📱 React-Native {#react-native}

[![Expo Router RFC - API 路由](/emails/issues/160/expo-api-routes.jpg)](https://blog.expo.dev/rfc-api-routes-cce5a3b9f25d)

[**Expo Router RFC - API 路由**](https://blog.expo.dev/rfc-api-routes-cce5a3b9f25d)

我期待已久！Expo 正透過 Expo Router 慣例提供 API 路由部署功能，進軍伺服器端領域。您可以自行部署這些路由，且很快就能在主流託管服務上部署，但很明顯，長期目標是將伺服器運行時託管在 EAS 上。

這標誌著 Expo 新時代的開端。在解決原生行動端問題後，現在是時候透過伺服器端渲染（SSR）和 React 伺服器元件（RSC）填補網頁支援的空白，使跨平台成為主流現實。
