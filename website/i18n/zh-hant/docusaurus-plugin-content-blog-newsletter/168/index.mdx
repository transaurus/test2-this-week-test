## ⚛️ React {#react}

[![TanStack Router 1.0](/emails/issues/168/tanstack-router.jpg)](https://tanstack.com/router/v1)

**[TanStack Router 1.0](https://tanstack.com/router/v1)**

備受期待的 TanStack Router 以穩定的 1.0 版本推出，並附帶一個精美的 [🐦 宣傳影片](https://twitter.com/tannerlinsley/status/1738709130213560497)，正好趕上聖誕節。這個新的路由在類型安全、URL 狀態管理（搜尋參數）和快取等方面，突破了可能的界限。

[SSR 指南](https://tanstack.com/router/v1/docs/guide/ssr) 提到了一個 `@tanstack/react-start` 套件，可能是即將推出的 [🐦 TanStack Start 框架](https://twitter.com/sebastienlorber/status/1742589454563369123) 的預告。

您可能錯過的其他近期 TanStack 套件：
