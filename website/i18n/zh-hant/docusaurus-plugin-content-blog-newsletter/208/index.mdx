## ⚛️ React {#react}

[![Sathya Gunasekaran - React Compiler 的未來發展方向](/emails/issues/208/compiler.jpg)](https://www.youtube.com/watch?v=qd5yk2gxbtg)

🎥 [**Sathya Gunasekaran - React Compiler 的未來發展方向**](https://www.youtube.com/watch?v=qd5yk2gxbtg)

我終於跟上這場精彩演講，內容不僅宣布 [React Compiler 進入 beta 階段](https://react.dev/blog/2024/10/21/react-compiler-beta-release)，更揭示了其他重要進展。該編譯器不僅實現自動記憶化，更將成為建構更多優化功能的平台。演講中展示了三項即將推出的功能：

- 類型配置：您可以透過提供額外類型資訊，協助編譯器更有效運作。
- JSX 內聯：編譯器可移除無效的 JSX 運行時調用，替換為預構建的 JSX 物件。`jsx('div')` 會轉為 `{type: 'div'}`。
- JSX 輪廓化：編譯器能自動提取子組件，使其可獨立優化。這在使用 `array.map` 和內聯 JSX 處理列表元素時特別有用，編譯器會建立 `<Item>` 組件（參見演講截圖，或[此 PR](https://github.com/facebook/react/pull/30956)說明）。

## 📱 React-Native {#react-native}

[![React Navigation 7.0](/emails/issues/208/reactnavigation7.jpg)](https://reactnavigation.org/blog/2024/11/06/react-navigation-7.0/)

[**React Navigation 7.0**](https://reactnavigation.org/blog/2024/11/06/react-navigation-7.0/)

React Navigation v7 已發布，將驅動即將到來的 Expo Router v4（隨 Expo SDK 52 推出，兩者均為測試版）。重點功能：

- 畫面預加載：可提前準備待導航畫面，提升感知效能。此功能極具潛力，未來可能整合 Concurrent React。
- 靜態 API：簡化導航器配置，提升 TypeScript 與深層連結的開發體驗。
