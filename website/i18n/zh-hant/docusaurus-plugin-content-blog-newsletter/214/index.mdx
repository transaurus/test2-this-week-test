**React 19 與自訂元素**

我發現幾篇相關有趣文章：

- 📜 [React 19 與 Web Component 範例](https://frontendmasters.com/blog/react-19-and-web-component-examples/)：Chris Coyier 透過建立 Lit 元素並用 JSX 渲染來測試 React 19 的自訂元素支援。可無縫傳遞物件作為 props，但事件處理器可能需要調度自訂事件。
- 📜 [React 19 全面支援自訂元素](https://aleks-elkin.github.io/posts/2024-12-06-react-19/)：比較 React 18 與 React 19 的支援改進。
- 📜 [甜蜜的 Suspense](https://plainvanillaweb.com/blog/articles/2024-09-09-sweet-suspense/)：有趣實驗，用自訂元素重建 React lazy/Suspense/ErrorBoundary。
- 📜 [同構 Web Components](https://jakelazaroff.com/words/isomorphic-web-components/)：雖未使用 React，但提供了 Web components 伺服器端渲染支援的實用概述。
