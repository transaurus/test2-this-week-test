[![Statically analysing React components for fun and profit](/emails/issues/163/react-forget.jpg)](https://www.youtube.com/watch?v=kjOacmVsLSE)

🎥 [**靜態分析 React 元件：從樂趣到效益**](https://www.youtube.com/watch?v=kjOacmVsLSE)

Sathya 正在開發期待已久的 React-Forget 編譯器，本次演講揭露了關鍵細節：

- 基於 React 新模型（不再使用類別或 `this`）實現編譯可行性
- 具備局部推理能力，可平行優化並逐步採用
- 支援無類型 JavaScript，並理解 React 慣例（如 hooks 及 "use" 前綴）
- 別名分析系統比 V8 引擎更完整

Sathya 表示未來幾週將有更多 React-Forget 消息，包含 2 天後在 [React Advanced 的《理解慣用 React》演講](https://reactadvanced.com/schedule-offline)。簡言之，我們已迫不及待想了解更多！
