## ⚛️ React {#react}

[![Astro 2.9](/emails/issues/153/astro.jpg)](https://astro.build/blog/astro-290/)

[**Astro 2.9**](https://astro.build/blog/astro-290/)

Astro 發布了新版本，包含期待已久的實驗性功能：[視覺轉場](https://developer.chrome.com/docs/web-platform/view-transitions/)。此功能可輕鬆創建 HTML 頁面間的流暢動畫轉場，無需依賴客戶端路由：不像 Next.js 或 Remix 需使用 `<Link>` 組件。對 React 開發者而言，這使框架在 UX 方面更具吸引力，能與我們慣用的傳統 SPA 模式競爭，同時保持更簡潔的模型。注意：這些轉場不會保留 React 狀態於導航間，這可能造成問題。請留意[視覺轉場的支援度](https://caniuse.com/?search=View%20Transition%20API)仍有限，但進展迅速，可視為漸進增強功能，且 Astro 提供降級模式。
