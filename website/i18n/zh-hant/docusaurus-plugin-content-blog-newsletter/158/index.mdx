## ⚛️ React {#react}

[![Astro 3.0](/emails/issues/158/astro3.jpg)](https://astro.build/blog/astro-3/)

[**Astro 3.0**](https://astro.build/blog/astro-3/)

此版本創新運用 View Transition API，實現頁面間流暢動畫導航並保留元件狀態，大幅降低多數場景對 SPA 的需求。

其他重大改進：

- 圖片優化功能穩定
- 渲染效能提升：Astro 元件渲染速度加快 30-75%
- 無伺服器 SSR 強化：新增主機平台整合方式
- JSX HMR 增強：支援 React 和 Preact 的 Fast Refresh
- 建置輸出優化：生成更簡潔高效的 HTML

## 📱 React-Native {#react-native}

[![VisionCamera 3.0](/emails/issues/158/visioncamera3.jpg)](https://github.com/mrousavy/react-native-vision-camera/releases/tag/v3.0.0)

[**VisionCamera 3.0**](https://github.com/mrousavy/react-native-vision-camera/releases/tag/v3.0.0)

VisionCamera 的新版本已正式發布。重點更新：

- 全面重寫 Android Camera2
- 自訂 GPU OpenGL 視訊管線
- 新增相機功能
- 全新幀處理引擎 (react-native-worklets-core)
- 新增幀處理插件 API

Marc 最終決定[移除 Skia 整合功能](https://github.com/mrousavy/react-native-vision-camera/pull/1740)，以保持 VisionCamera 的程式碼體積與複雜度在合理範圍。

在[**使用 VisionCamera V3、TFLite 和 Skia 進行姿態偵測**](https://mrousavy.com/blog/VisionCamera-Pose-Detection-TFLite)一文中，他詳述了此類整合所能實現的功能，成果令人驚豔。
