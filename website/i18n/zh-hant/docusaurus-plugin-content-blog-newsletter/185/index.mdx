## ⚛️ React {#react}

[![使用 useDeferredValue 實現流暢的 UI 優化](/emails/issues/185/useDeferredValue.jpg)](https://www.joshwcomeau.com/react/use-deferred-value/)

**[使用 useDeferredValue 實現流暢的 UI 優化](https://www.joshwcomeau.com/react/use-deferred-value/)**

Josh Comeau 帶來一篇精彩文章，展示 `useDeferredValue()` hook 的實用性，該功能讓您能透過將高成本元件渲染降級為低優先級任務，選擇性啟用 Concurrent React。他詳細說明實際遇到的效能問題，以及該 hook 如何協助解決。要高效運作，關鍵是使用 `memo()` 包裹高成本元件。互動式時間軸有助理解其與節流（throttling）的差異。

[![為何選擇 React Query？](/emails/issues/185/reactquery.jpg)](https://ui.dev/why-react-query)

**[為何選擇 React Query？](https://ui.dev/why-react-query)**

Dominik Dorfmeister 推出官方 React Query 課程，並免費發布首堂課程。這是一篇精彩的互動式文章，逐步展示 React Query 的完整功能，包括處理競態條件、請求去重複化與快取機制。它功能全面，更像是一個「非同步狀態管理器」而不僅是數據獲取庫。即使不使用 React Query，也值得所有 React 開發者一讀。
