[![Vite 4](/emails/issues/128/vite4.png)](https://vitejs.dev/blog/announcing-vite4.html)

**[Vite 4](https://vitejs.dev/blog/announcing-vite4.html)**

Vite 是前端工具，其普及度持續增長。此版本採用 Rollup 3，大幅簡化了內部資源管理。現提供兩種具有不同權衡的 React 插件：

- [vite-plugin-react](https://github.com/vitejs/vite-plugin-react)：esbuild + Babel
- [vite-plugin-react-swc](https://github.com/vitejs/vite-plugin-react-swc)：在開發環境用 SWC 取代 Babel，可改善冷啟動和熱模組替換速度

[![Codux - React 專案視覺化 IDE](/emails/issues/128/codux.jpg)](https://dev.to/codux/introducing-codux-15j5)

**[Codux - React 專案視覺化 IDE](https://dev.to/codux/introducing-codux-15j5)**

Nadav 是 Wix 的共同創辦人，他向我們介紹了他的新專案，一個用於 React（以及即將支援其他框架）的視覺化整合開發環境（IDE）。這並不是要取代你目前的 IDE，而是要補足它。Codux 讓你能夠獨立地處理一個元件，包括透過視覺化介面編輯屬性和樣式。影片 [Meet Codux](https://www.youtube.com/watch?v=KG6i6cCz0P0) 應該能幫助你更了解。它有點像 Storybook，但具備編輯能力，並且能與你的程式碼進行雙向同步。
