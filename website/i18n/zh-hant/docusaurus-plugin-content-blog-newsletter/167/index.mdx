## 📱 React-Native {#react-native}

[![Unistyles 2.0](/emails/issues/167/unistyles20.jpg)](https://www.youtube.com/@notjustdev/live?sub_confirmation=1)

[**歡迎使用 Unistyles 2.0！**](https://www.reactnativecrossroads.com/posts/unistyles-2-preview)

Unistyles 是一款跨平台樣式庫，作為 StyleSheet 的超集構建（若您使用 React Native，可能熟悉 StyleSheet.create...），類似 TypeScript 之於 JavaScript 的關係。該庫於三個月前首次發布，已在 GitHub 獲得超過 500 顆星，現正邁向第二個主要版本：強化 TypeScript 型別、不再依賴 React Context，並採用全新 C++ 綁定以提升效能。最引人注目的新功能是插件支援，有望開啟創新應用場景。
